# As-Is: 품질검사 프로세스 분석

> 본 프로세스는 현대모비스 내부 자료가 아닌, 공개된 M-SPHERE 2024 발표자료 및
> 전장부품 제조업 표준 공정을 근거로 합리적으로 추정(Assumption)한 것이며, 구체적 출처는 '가정 및 한계'에서 확인 가능함.

## 전체 생산 흐름 (개요)

​```mermaid
flowchart LR
    A[부품 입고/자재검수] --> B[SMT 표면실장]
    B --> C[조립]
    C --> D[품질검사]
    D --> E[포장/출하]
​```

## 품질검사 단계 세부 흐름 (Swimlane)

​```mermaid
flowchart TD
    subgraph 생산라인
        A1[조립 완료 제품 이동]
    end
    subgraph 검사팀
        B1[1차 외관검사 - 육안]
        B2[2차 기능검사 - 전기적 특성]
        B3[샘플링 정밀검사]
        B4[검사 결과 기록]
    end
    subgraph 품질관리팀
        C1[불량 데이터 집계]
        C2[불량 원인 분석]
        C3[라인 피드백]
    end

    A1 --> B1 --> B2 --> B3 --> B4
    B4 --> C1 --> C2 --> C3
    C3 -.재작업/조정.-> A1
