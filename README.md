# PwC_AX-Advisory-prep
삼일PwC AX Advisory 입사지원을 위한 프로젝트 - 프로세스 분석 및 솔루션 제안
## 프로젝트 개요
삼일PwC AX Advisory 부문 지원을 위한 자기주도 프로젝트
비즈니스 프로세스 분석을 통해 Pain Point를 도출하고,
Digital/AI/Automation 기반 개선 방안을 제안하는 과정을 기록할 예정입니다.

## 목표
- 특정 산업/기업의 As-Is 프로세스 분석
- Pain Point 및 개선 기회 도출
- AI, Data, Automation 기술 매핑
- 데이터 기반 인사이트 도출
- 최종 Advisory 제안서(PPT) 작성

## 진행 방법론
1. 산업/기업 리서치
2. 프로세스 분석 (Swimlane / Flow chart)
3. Pain Point 도출 (GitHub Issues로 관리)
4. Digital/AI 솔루션 매핑
5. 데이터 분석 실습
6. 최종 제안서 작성

## 폴더 구조
- `01_research` : 산업/기업 리서치 자료
- `02_process-analysis` : 프로세스 분석 자료
- `03_pain-points` : Pain Point 정리
- `04_solution-mapping` : 솔루션 매핑
- `05_data-analysis` : 데이터 분석
- `06_deliverable` : 최종 산출물
